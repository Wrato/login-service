# Login Service

Microservicio para **registro, autenticación y consulta de usuarios** usando Spring Boot, JPA y JWT.

---

## Tecnologías

- Java 11
- Spring Boot 2.5.14
- H2 Database (runtime)
- Spring Security Crypto (BCrypt)
- JWT (jjwt)
- ModelMapper
- JUnit 5 + Mockito
- JaCoCo (cobertura de pruebas)
